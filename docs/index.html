    order 

    @function order($position) { ... }

    Description

    Provides a built in, editable list for syntactic sugaring of z-indices. The current list for Siyavula contains "tertiary", "secondary", "primary" and "over9000", with each successive entry having a higher z-index than the previous one.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $position

    Which position in the list to use

    		 String — none

    Returns

    The resultant z-index

    Number

    Example

    
$list: tertiary, secondary, primary;
.foo{ z-index: order(primary)} //returns 3

    Author

    • Sara Soueidan

    convert 

    @function convert($value, $curr-unit, $conv-unit) { ... }

    Description

    Converts one amount of a typographic unit into the equivalent amount in another unit

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $value

    The value to convert

    		 Number — none
    $curr-unit

    The unit to convert from. Support for px, ems, pts and percent

    		 String — none
    $conv-unit

    The unit to convert to. Support is the same, sans pts

    		 String — none

    Returns

    The converted typographic unit

    Any

    Example

    
.foo{ font-size: convert( 16, px, percent ); }
//will return
.foo{ font-size: 100%; }

    breakpoint 

    @mixin breakpoint($breakpoint) { ... }

    Description

    Adds syntactic sugar to media query declarations. Instead of declaring unit values you're able to declare things like "small", "medium", "large" etc.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $breakpoint

    Which breakpoint to use

    		String — none

    Example

    
.foo{
 @include breakpoint(small){
   width: 100%;
 }
}
//will return
  @media all and (min-width: 480px){
    .foo{
      width: 100%;
    }
  }

    Author

    • Hugo Giraudel
